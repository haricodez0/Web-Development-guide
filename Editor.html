Editors

All you need for learning and using HTML is a notepad, which is a simple text editor

Learn HTML Using Notepad or TextEdit

Web pages can be created and modified by using professional HTML editors.

However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).

Using a simple Text Editor is a good way to learn HTML

Professional HTML editors such as Visual Studio Code is used for Professional Software Development which makes your coding more customized and easy

Follow the steps below to create your first web page with Notepad or TextEdit.

Step 1: Open Notepad(For Windows)

Click the Windows icon and search for Notepad and open it

Step 1: Open TextEdit(For Mac).

Open Finder > Applications > TextEdit.

Also change some preferences to get the application to save files correctly. In Preferences > Format > choose ''Plain Text'.

Then under 'Open and Save', check the box that says 'Display HTML files as HTML code instead of formatted text'.

Then open a new document to place the code.

Write the HTML

Write or copy the following HTML code into Notepad:

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>

Save the HTML file

After writing the HTML you need to save it.

For it to act as an HTML file we need to save it in a specific way.

First Select File > Save as in the Notepad menu.

Name the file 'index.html'

and set the encoding to UTF-8 (which is the preferred encoding for HTML files)

You can save the file as either .htm or .html both works perfectly fine

View your HTML file in a Browser

Double click the file to open it.

If you want to choose what Browser you want to view it in, then right click and click on 'Open with'
