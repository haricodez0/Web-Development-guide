Basics

In this Page you will learn about all the basic HTML tags

HTML Document

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The <!DOCTYPE html> declaration represents the document type, and helps browsers to display web pages correctly.

The HTML document itself begins with <html> and ends with </html>>.

The part between <body> and </body> is what will be shown in the webpage.

Headings

HTML headings tag ranges from <h1> to <h6>.

The lower the number the bigger the font size.

Which means the <h1> tag is used for something important such as the heading or title.

While the others for less important headings.

<h1>This is a Heading</h1>

Paragraphs

HTML paragraphs are defined using the <p> tag.

You can type anything in the body and it will show up as text but the <p> is a better option since it allows you to format and style

<p>This is a Paragraph</p>

Links

HTML links are defined as <a> tag

The link's destination is specified in the href attribute

Attributes are used to provide additional information about HTML elements.

You will learn deeper about attributes in the next page

<a href="Google.com">This is a Link</a>

Images

HTML images are defined with the <img>

You can get the image by entering the file location in the src attribute.

<img src="Images/Icon.png">This is a Link</img>
